## L'ascension : de nonprofit à 150 milliards

**Fondée en 2015** par Sam Altman, Elon Musk, Ilya Sutskever et d'autres. Organisée comme nonprofit. Mission : créer une IA sûre pour l'humanité.

**Problème** : entraîner des modèles coûte des milliards. Un nonprofit ne peut pas lever ça.

**2019** : crée une entité "capped-profit". Profit illimité avec un plafond d'investisseur. Contrevérité ? Peut-être. Mais levé les milliards. Microsoft partenaire stratégique (10+ milliards d'investissement).

**2024** : 150+ milliards de valorisation. La startup est devenue le leader de facto.

<Callout type="info" title="Le problème structural d'OpenAI">
Nonprofit parie l'IA sûre est son objectif. Mais a créé une structure for-profit pour lever de l'argent. Est-ce qu'une structure for-profit peut vraiment prioritariser la sécurité ? Les critiques disent non. OpenAI argumente oui.
</Callout>

## La famille GPT

### GPT-3 et 3.5

**GPT-3** (2020) : 175 milliards de paramètres. Première démonstration que le scaling fonctionne pour les LLMs. Vous augmentez la taille, la performance augmente. Révolutionnaire.

**GPT-3.5 et ChatGPT** (novembre 2022) : le moment clé. ChatGPT atteint **100 millions d'utilisateurs en 2 mois**. Le lancement grand public le plus rapide de l'histoire. L'IA devient mainstream.

### GPT-4 (mars 2023)

**Saut qualitatif**.

- **Multimodal** : accepte texte et images
- **Performances** : passe le barreau dans le top 10%. Réussit les examens universitaires.
- **Contexte** : 32K puis 128K tokens. Lisez un livre entier.
- **Fiabilité** : moins d'hallucinations que GPT-3.5

### GPT-4o et 4o mini (2024)

**GPT-4o** : "omni" = multimodal natif. Texte, image, audio. Latence basse.

**GPT-4o mini** : compacte, économique. Pour apps à large échelle.

<Callout type="tip" title="Le modèle économique d'OpenAI">
OpenAI a inventé le freemium pour les LLMs : ChatGPT gratuit (limited), ChatGPT Plus (20$/mois), API au token. Tout le monde a copié. C'est devenu la norme.
</Callout>

## La série o : le changement de paradigme

### o1 (septembre 2024)

O1 marque un tournant : stop de juste augmenter la taille.

Nouveau : **compute at inference time**. Dépensez du calcul en réfléchissant, pas en entraînement.

- **Chain-of-thought interne** : le modèle réfléchit avant de répondre
- **Performances** : rivalise avec les doctorants en physique, maths, biologie
- **AIME** : 83% sur les compétitions math américaines

Le bond : GPT-4 (2-4%) → o1 (83%). Massive.

### o3 et o4-mini (2025)

**o3** : amélioration majeure du raisonnement. Records sur ARC-AGI.

**o4-mini** : version compacte et rapide.

**Paradigme clé** : scaling du temps de raisonnement = aussi important que scaling des paramètres.

## Sora : la génération vidéo

**Annoncée février 2024**.

Génère des vidéos 60 secondes à partir de texte. Comprend la physique (mouvements, réflexions, interactions). Architecture diffusion Transformer sur patches spatio-temporels.

Déclenché une course : Runway Gen-3, Google Veo, Kling. Et des débats : implications pour les créatifs.

## Les autres produits

**DALL-E 3** : images, intégré à ChatGPT.

**Whisper** : reconnaissance vocale open-source. Gratuit. Haute qualité.

**Codex / GitHub Copilot** : programmation. Partenariat GitHub.

**ChatGPT Plugins / GPTs** : écosystème d'extensions. Agents personnalisés.

## Les controverses

OpenAI fait face à beaucoup.

**Ilya Sutskever** (fondateur, chief scientist) a quitté en 2024. Raisons non claires.

**Crise du board** (novembre 2023) : Sam Altman a été viré puis réembauché. Drama.

**Copyright lawsuits** : New York Times poursuit OpenAI pour usage du contenu.

**Manque de transparence** : accusations sur la sécurité, le processus d'entraînement, l'AGI timeline.

**Abandon de la mission nonprofit** : critiques que OpenAI a oublié le nonprofit pour le profit.

<Callout type="warning" title="Les tensions structurelles">
OpenAI a une mission (IA sûre pour l'humanité) et une structure (for-profit maximisant le profit). Ces deux peuvent conflictuer. Un modèle par-dessus le autre, et ça craque. Seul le temps dira si OpenAI peut réconcilier les deux.
</Callout>

## Position : leader mais pas inarrêtable

OpenAI reste le leader : ChatGPT le produit le plus utilisé. L'API standard de facto. Les talents meilleurs.

Mais la concurrence monte : Anthropic (Claude), Google (Gemini), Meta (LLaMA open), DeepSeek (IA chinoise compétitive).

OpenAI a gagné round 1. Round 2 ? À déterminer.
