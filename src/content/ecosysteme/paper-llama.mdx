## L'égalisateur

LLaMA 13B vs GPT-3 (175B). Même performance.

13 milliards vs 175 milliards.

Comment ? Données. Scaling. Chinchilla.

LLaMA l'applique. Entraîne 13B sur 1 trillion de tokens. GPT-3 ? 175B sur 300 milliards.

<Callout type="info" title="L'impact : l'open-source devient réel">
Avant LLaMA, les meilleurs LLMs étaient propriétaires (OpenAI, Google, Meta). LLaMA rend open-source viable. Des milliers de chercheurs expérimentent. L'innovation s'accélère. L'industrie change.
</Callout>

## Les données publiques uniquement

Meta utilise seulement du public :

- Common Crawl (67%) : pages web filtrées
- C4 (15%)  : Common Crawl nettoyé
- GitHub (4.5%) : code
- Wikipedia (4.5%)
- Books et ArXiv : plus

Pas de données secrètes. Pas de Web-at-scale propriétaire. C'est important pour la reproductibilité.

## Les choix architecturaux

LLaMA n'invente rien.

Combine les best practices :

- **Pre-norm** (RMSNorm) : normalise avant chaque sous-couche
- **SwiGLU** : meilleure activation que ReLU
- **Rotary Positional Embeddings (RoPE)** : encodage positionnel
- **Pas de bias** : courche linéaires sans bias

Cette recette : devient le standard. Mistral, LLaMA 2/3, Gemma. Tous la copient.

## Résultats

| Modèle | Paramètres | Tokens | Performance |
|--------|-----------|--------|-------------|
| LLaMA-7B | 7B | 1T | Surpasse GPT-3 |
| LLaMA-13B | 13B | 1T | Surpasse GPT-3 |
| LLaMA-65B | 65B | 1.4T | Rivalise Chinchilla/PaLM |

Le proof : la taille ne rule pas. Les données règnent.

## La fuite et ses conséquences

LLaMA était réservé aux chercheurs. Poids leaked.

Quelques jours après. Partout sur internet.

Meta : surpris. Ou peut-être pas contrariée ?

La fuite a accéléré l'adoption. Tout le monde a accès.

## L'écosystème des dérivés

**Alpaca** : LLaMA-7B fine-tuned avec instructions GPT-4. Coût : 600 dollars. Performances proches de GPT-3.5. Révélation.

**Vicuna** : fine-tuned sur conversations ShareGPT.

**WizardLM** : Evol-Instruct (instruction complexity evolution).

**Code Llama** : spécialisé code.

Douzaines d'autres variants. LLaMA est la fondation.

## L'évolution

**LLaMA 2** (juillet 2023) : licence commercial. 7B/13B/70B.

**LLaMA 3** (avril 2024) : 8B/70B. Contexte 8K. Performances excellentes.

**LLaMA 3.1** (juillet 2024) : 8B/70B/405B. Contexte 128K.

**LLaMA 4** (2025) : MoE architecture.

Chaque génération : meilleure. LLaMA remain king of open-source.

<Callout type="paper" title="Le tournant de l'industrie">
LLaMA a démontré que l'open-source est viable pour les LLMs de pointe. Avant, monopole propriétaire. Après, écosystème ouvert florissant. La structure industrielle de l'IA a changé.
</Callout>

<Callout type="warning" title="Open-weight, pas open-source au sens strict">
LLaMA : poids publics. Pas les données d'entraînement. Pas le code complet. Pas l'infrastructure. Reproductibilité partielle. Des projets comme OLMo (Allen AI) vont plus loin.
</Callout>
