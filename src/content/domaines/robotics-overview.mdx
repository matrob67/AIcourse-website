## La convergence : 50 ans d'évolution séparée, soudain ensemble

Pendant des décennies, robotique et IA générative en parallèle. Robots industriels : trajectoires fixes programmées. Modèles de langage : compréhension/génération texte, zéro connexion monde physique.

Depuis 2022-2023 : révolution. **Foundation models** — ces géants pré-entraînés sur d'immenses corpus — appliqués directement à robotique. **Idée centrale** : transférer compréhension du monde (langage, vision, raisonnement) vers contrôle de robots physiques.

<Callout type="info" title="Qu'est-ce qu'un Foundation Model robotique ?">
Un foundation model robotique = modèle pré-entraîné capable de comprendre instructions en français, percevoir environnement via vision, générer actions motrices. Unifie perception + raisonnement + action dans seul système.
</Callout>

## Pourquoi ça change tout : le problème historique

Avant : robotique souffrait de limitations majeures :
- **Généralisation limitée** : robot entraîné saisir tasses ne savait saisir bouteilles
- **Données rares/coûteuses** : collecter démos robotiques nécessite hardware physique + temps
- **Pas compréhension sémantique** : robot ne savait pas QUOI manipulait, seulement COMMENT
- **Fragilité** : moindre changement environnement échouait système

**Foundation models résolvent plusieurs :**
- **Compréhension langage** : robot peut recevoir instructions français naturel
- **Connaissance du monde** : LLM absorbé immense quantité connaissances objets/propriétés/usages
- **Raisonnement** : décomposer tâche complexe en sous-tâches
- **Vision généraliste** : reconnaître milliers objets sans entraînement spécifique

<Callout type="tip" title="L'analogie NLP">
NLP révolution avec BERT/GPT : modèles spécialisés → généralistes. Robotique suit même chemin, quelques années en retard. Foundation models = transformation robotique en cours.
</Callout>

## Approches principales



**Vision-Language-Action (VLA)** : image + instruction textuelle → actions robot directement. **RT-2** Google : exemple emblématique. Prendre VLM pré-entraîné, fine-tuner sur données robotiques.

**LLM comme planificateur haut niveau** : LLM décompose instruction complexe en étapes simples, délègue à contrôleurs spécialisés bas niveau. **SayCan** Google, **Code as Policies**.

**World Models** : apprendre modèle interne de physique monde. Robot **simule mentalement** conséquences actions avant exécution. Excellent pour planification long terme.

## Défis majeurs

**Fossé sim-to-real** : modèles pré-entraînés en simulation (Isaac Sim, MuJoCo). Transfert vers monde réel reste majeur. Textures, physique, imprécisions réalité diffèrent considérablement simulation.

**Sécurité** : LLM qui hallucine = texte incorrect. Robot qui hallucine = dommages physiques. Fiabilité et sécurité foundation models en robotique = enjeux critiques.

**Latence** : modèles grands sont lents vs boucles contrôle robotique (souvent 100+ Hz). Faire tourner milliards paramètres temps réel sur robot embarqué = défi ingénierie.

<Callout type="warning" title="Maturité technologique">
Malgré progrès spectaculaires, foundation models robotique = stade recherche. Aucun robot commercial grand public repose entièrement sur VLA. Démos impressionnantes. Passage à l'échelle industrielle = reste à faire.
</Callout>

## Les acteurs clés

**Google DeepMind** : RT-1, RT-2, RT-X, SayCan, pionniers. **Tesla** : Optimus, approche end-to-end FSD. **Figure AI** : humanoïdes + OpenAI. **1X Technologies** : NEO, financé OpenAI. **NVIDIA** : Isaac Sim + GR00T. **Meta** : open-source + recherche manipulation.

## Perspectives

Robotique vit moment comparable arrivée ImageNet/CNN pour vision. Ingrédients présents : modèles puissants, simulation scalable, investissements massifs. Prochaines années détermineront si foundation models donnent robots flexibilité et intelligence pour opérer monde réel.

Le parallèle ImageNet : avant 2012, vision par ordinateur lent, fragile. ImageNet + CNN = révolution. Nous sommes au même point en robotique avec foundation models.
