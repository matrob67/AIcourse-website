## De la compréhension à l'action : le saut conceptuel

Pendant 70 ans, la robotique et le langage naturel = domaines séparés. Un robot n'avait pas à comprendre « apporte-moi un verre d'eau ». Il exécutait un programme prédéfini : « va à la cuisine → saisir verre → revenir ».

**RT-2** (Robotic Transformer 2, Google DeepMind, 2023) fait le saut : prend un modèle vision-langage pré-entraîné sur **données web** (des milliards d'images et textes) et le transforme en contrôleur robotique.

<ArxivLink id="2307.15818" title="RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control (Brohan et al., 2023)" />

Au lieu de construire un système robotique from scratch, RT-2 **transfère les connaissances massives** acquises par le VLM.

<Callout type="info" title="Les actions comme tokens">
Innovation clé : représenter actions du robot comme tokens de texte. Coordonnées (x, y, z), rotations, commandes préhension = discrétisées et ajoutées au vocabulaire. Générer une action = générer du texte. Le modèle sait déjà le faire.
</Callout>

## L'architecture VLA

**Entrée vision** : image de l'environnement via caméra du robot.
**Entrée langage** : instruction textuelle (ex : « déplace la canette vers le personnage de dinosaure »).
**Sortie action** : commandes motrices du robot, tokenisées en texte.

**Modèles de base** : **PaLI-X** (55B paramètres) ou **PaLM-E** (12B). Fine-tuning sur données robotiques pour apprendre générer actions.

## Le transfert magique : connaissances web → robotique

Ce qui rend RT-2 révolutionnaire : capacité à **exploiter connaissance web** pour tâches robotiques jamais vues en entraînement :

- **Raisonnement sémantique** : « mets l'objet dans la bonne poubelle » → robot comprend papier = recyclage
- **Reconnaissance symboles** : identifier logos, drapeaux, célébrités sur objets
- **Raisonnement mathématique** : « déplace vers le nombre le plus petit » fonctionne sans entraînement spécifique
- **Chaîne de raisonnement** : décomposer instructions complexes

<Callout type="tip" title="Chain-of-thought robotique">
RT-2 peut utiliser chain-of-thought : avant générer action, « réfléchit » en français naturel. Exemple : « ramasse un objet pour écrire » → identifie objets → raisonne choisir stylo → génère action.
</Callout>

## Généralisation zéro-shot

Sur tâches de raisonnement sémantique (**jamais vues en entraînement**), RT-2 atteint 62% succès vs 32% pour RT-1. Prouverait que transfert de connaissances depuis pré-entraînement web **fonctionne réellement**.

## De RT-1 à RT-2 : l'évolution

**RT-1** (2022) : modèle robotique entraîné uniquement sur données démos robotiques. Performant sur tâches vues, généralisation faible. Pas compréhension sémantique.

**RT-2 améliorations** :
- Pré-entraînement web : connaissance monde de milliards pages
- Échelle : 35M (RT-1) → 55B paramètres
- Raisonnement : capacité symbolique/sémantique absente RT-1
- Co-fine-tuning : entraînement simultané web + robotique, évitant l'oubli catastrophique

## RT-X et Open X-Embodiment

Après RT-2, Google lance **Open X-Embodiment** (RT-X) : effort collaboratif regroupant données 22 types de robots différents de 21 institutions. Objectif : dataset unifié pour entraîner modèles robotiques généralistes, image d'ImageNet pour vision.

<Callout type="paper" title="Impact scientifique">
RT-2 démontré **première fois** qu'un modèle vision-langage pouvait être directement transformé en contrôleur robotique, avec transfert de connaissances mesurable depuis web. Paradigme VLA devenu référence recherche robotique.
</Callout>

## Limitations

- **Latence** : 55B paramètres trop lent pour contrôle temps réel haute fréquence
- **Robot unique** : démontré uniquement sur robot mobile Google (Everyday Robots)
- **Données propriétaires** : modèle et données non publics
- **Tâches simples** : démos limitées manipulation simple sur table

Malgré limitations, RT-2 ouvert la voie : OpenVLA, Octo, et tous les grands labos travaillent maintenant sur VLA. L'influence RT-2 transforme robotique.
