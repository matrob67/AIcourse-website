## 3 millions d'articles par an : impossible à lire

Chaque année, 3+ millions d'articles scientifiques publiés. Un chercheur ne peut lire qu'infime fraction de sa spécialité. **LLMs scientifiques** visent synthétiser, organiser, rendre accessible cette montagne de connaissances.

## Galactica (Meta AI, novembre 2022)

**120 milliards paramètres**, entraîné sur corpus scientifique :
- 106 milliards tokens : 48M articles scientifiques, manuels, Wikipedia, StackExchange
- Bases de données : PDB (protéines), UniProt (gènes)
- Tokenisation spéciale : séquences aminées, SMILES moléculaires, formules LaTeX = tokens spéciaux

**Capacités annoncées** : résumé articles, génération texte scientifique avec citations, prédiction propriétés moléculaires, raisonnement mathématique, annotation séquences protéiques.

### La controverse

Retiré de démo publique 3 jours après lancement. Pourquoi ? Le modèle générait contenu scientifique **plausible mais factuellement incorrect** avec grande confiance :
- Articles faux avec vraies citations mélangées à inventées
- Reproduction biais littérature scientifique
- Craintes faciliter fraude scientifique

<Callout type="warning" title="Le problème Galactica">
Le retrait illustre risque fondamental : un modèle générant texte scientifique fluide et confiant peut tromper même experts si erreurs factuelles subtiles. Fluence linguistique ≠ exactitude scientifique. C'est le « bullshit problem » appliqué à la science.
</Callout>

## SciBERT et les modèles encodeurs

**SciBERT** : BERT pré-entraîné sur littérature scientifique.
- Corpus : 1.14M articles Semantic Scholar (82% biomédical, 18% informatique)
- Vocabulaire : WordPiece spécifique domaine scientifique
- Taille : BERT-Base (110M paramètres)
- Applications : NER scientifique, classification articles, extraction relations

SciBERT surpasse BERT sur tâches NLP scientifiques grâce à vocabulaire adapté et représentations contextuelles entraînées sur texte scientifique.

**Variantes** : **PubMedBERT** (exclusivement PubMed/biomédical), **BioBERT**, **MatSciBERT** (science matériaux), **ScholarBERT** (75B tokens multi-domaines).

<Callout type="tip" title="Encodeurs vs Décodeurs">
Modèles encodeurs (BERT-like) = extraction/classification (NER, extraction relations, classification articles). Modèles décodeurs (GPT-like) = génération (résumé, Q&A, rédaction). Choix dépend application.
</Callout>

## Autres LLMs scientifiques

**LLEMMA** (7B/34B) : open-source, mathématiques, basé Code Llama, entraîné Proof-Pile-2.

**SciGLM** : bilingue (chinois-anglais), raisonnement scientifique.

**Semantic Scholar** (Allen AI) : moteur recherche IA littérature scientifique. **S2ORC** : corpus 81M articles texte intégral structuré.

**SPECTER** : embedding documents scientifiques pour recherche similarité.

**GPT-4 et Claude généralistes** : utilisés croissant par chercheurs pour assistance rédaction, revue littérature, explication concepts, debug code scientifique, génération hypothèses.

## Applications concrètes

**Extraction info scientifique** : NER scientifique (gènes, protéines, maladies, composés), extraction relations (« gène X associé maladie Y »), knowledge graphs.

**Recommandation articles** : modèles embedding permettent recommandation articles pertinents, détection travaux liés non cités, identification lacunes littérature.

**Revue par les pairs assistée** : outils commencent assister peer review : vérification cohérence résultats, détection erreurs statistiques, suggestion références manquantes.

<Callout type="paper" title="Le dilemme éthique">
Utilisation IA en recherche soulève questions : faut-il déclarer utilisation LLM ? Article co-écrit IA = original ? Principales revues (Nature, Science) ont établi guidelines, mais consensus évolue rapidement. Débat ongoing.
</Callout>

## Perspectives

LLMs scientifiques évoluent vers multimodalité : comprendre figures, tableaux, équations, données expérimentales. Objectif ultime : assistant recherche IA capable lire littérature, formuler hypothèses, concevoir expériences, analyser résultats — véritable partenaire recherche.
