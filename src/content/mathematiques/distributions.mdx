## Les distributions : le vocabulaire des modèles

Un modèle de machine learning, au fond, est une machine à produire des distributions de probabilité. Un classifieur produit une distribution sur les classes. Un générateur d'images produit une distribution sur les pixels. Un language model produit une distribution sur les tokens. Comprendre les distributions fondamentales, c'est comprendre le langage dans lequel les modèles s'expriment.

## Variables aléatoires : discrètes vs continues

Avant de parler de distributions, il faut distinguer deux types de variables aléatoires.

### Variable discrète

Elle prend un nombre fini ou dénombrable de valeurs. Exemples : le résultat d'un dé (1 à 6), la classe prédite par un classifieur (chat, chien, oiseau), le prochain token dans un vocabulaire de 50 000 mots.

On la décrit par une fonction de masse : $P(X = x)$ pour chaque valeur $x$ possible, avec $\sum_x P(X = x) = 1$.

### Variable continue

Elle prend des valeurs dans un intervalle réel. Exemples : la température, la valeur d'un pixel, un poids dans un réseau de neurones.

On la décrit par une densité de probabilité $f(x)$, avec $\int_{-\infty}^{+\infty} f(x) \, dx = 1$. Attention : $f(x)$ n'est pas une probabilité ! Elle peut dépasser 1. C'est l'intégrale sur un intervalle qui donne une probabilité.

## Distribution de Bernoulli

### Le pile ou face

La distribution la plus simple : deux résultats possibles, avec probabilité $p$ pour le "succès" et $1-p$ pour l'"échec".

$$P(X = 1) = p, \quad P(X = 0) = 1 - p$$

On écrit $X \sim \text{Bernoulli}(p)$.

### En ML : classification binaire

Chaque neurone de sortie d'un classifieur binaire modélise une Bernoulli. La fonction sigmoïde $\sigma(z) = \frac{1}{1 + e^{-z}}$ transforme un logit $z$ en paramètre $p \in [0, 1]$ :

$$P(y = 1 | x) = \sigma(f_\theta(x))$$

La binary cross-entropy loss découle directement de la log-vraisemblance d'une Bernoulli.

## Distribution catégorielle

### Le dé à N faces

Généralisation de la Bernoulli à $K$ catégories. On a un vecteur de probabilités $(p_1, p_2, \ldots, p_K)$ avec $\sum_{k=1}^K p_k = 1$.

$$P(X = k) = p_k$$

### En ML : classification multi-classe

Un classifieur ImageNet qui distingue 1 000 classes produit une distribution catégorielle à chaque prédiction. Un language model avec un vocabulaire de 50 000 tokens produit une catégorielle à 50 000 catégories pour chaque position.

La cross-entropy loss pour la classification multi-classe est exactement la log-vraisemblance négative d'une distribution catégorielle.

## La fonction softmax

### Transformer des logits en probabilités

Les couches de sortie d'un réseau produisent des logits : des nombres réels quelconques $z_1, z_2, \ldots, z_K$. Pour en faire une distribution catégorielle valide, il faut des nombres positifs qui somment à 1. C'est le rôle de softmax :

$$\text{softmax}(z_i) = \frac{e^{z_i}}{\sum_{j=1}^K e^{z_j}}$$

Chaque logit est exponentié (ce qui le rend positif), puis normalisé par la somme (ce qui garantit que le total vaut 1).

### Propriétés importantes

- **Préserve l'ordre** : si $z_i > z_j$, alors $\text{softmax}(z_i) > \text{softmax}(z_j)$
- **Sensible aux écarts** : l'exponentielle amplifie les différences. Un logit légèrement plus grand que les autres "gagne" une part disproportionnée de la probabilité
- **Invariance par translation** : $\text{softmax}(z + c) = \text{softmax}(z)$ pour toute constante $c$ (utile pour la stabilité numérique)

### Température

En divisant les logits par un facteur $T$ (la temperature) avant softmax, on controle la "forme" de la distribution :

$$\text{softmax}(z_i / T) = \frac{e^{z_i / T}}{\sum_{j=1}^K e^{z_j / T}}$$

- $T \to 0$ : la distribution devient un one-hot (argmax), le modèle est "confiant"
- $T = 1$ : comportement par défaut
- $T \to \infty$ : la distribution tend vers l'uniforme, le modèle est "indécis"

Ce paramètre est omniprésent dans la génération de texte avec les LLM et dans la distillation de connaissances.

## Distribution gaussienne (normale)

### La courbe en cloche

La distribution la plus importante en statistique et en ML. Elle est définie par deux paramètres : la moyenne $\mu$ et la variance $\sigma^2$ :

$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x - \mu)^2}{2\sigma^2}\right)$$

On écrit $X \sim \mathcal{N}(\mu, \sigma^2)$.

### Pourquoi elle est partout

- **Théorème central limite** : la somme de nombreuses variables aléatoires indépendantes tend vers une gaussienne, quelle que soit leur distribution d'origine. Comme les données réelles résultent souvent de multiples facteurs, elles suivent souvent approximativement une loi normale.
- **Entropie maximale** : parmi toutes les distributions ayant une moyenne et une variance données, la gaussienne est celle de plus grande entropie. C'est le choix le plus "neutre" quand on ne sait rien d'autre.
- **Propriétés algébriques** : somme de gaussiennes = gaussienne. Conditionnement d'une gaussienne = gaussienne. Cela simplifie énormément les calculs.

### En ML

- Les poids des réseaux sont souvent initialisés selon une gaussienne
- La régularisation L2 correspond à un prior gaussien sur les poids
- Les VAE supposent un prior gaussien dans l'espace latent
- Le bruit ajouté dans les modèles de diffusion est gaussien

## Gaussienne multivariée

### Quand on a plusieurs dimensions

Pour un vecteur aléatoire $\mathbf{x} \in \mathbb{R}^d$, la gaussienne multivariée est paramétrée par un vecteur moyenne $\boldsymbol{\mu} \in \mathbb{R}^d$ et une matrice de covariance $\boldsymbol{\Sigma} \in \mathbb{R}^{d \times d}$ :

$$f(\mathbf{x}) = \frac{1}{(2\pi)^{d/2} |\boldsymbol{\Sigma}|^{1/2}} \exp\left(-\frac{1}{2} (\mathbf{x} - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu})\right)$$

### Ce que contient la matrice de covariance

- **Diagonale** : les variances de chaque dimension (combien chaque variable varie)
- **Hors diagonale** : les covariances entre dimensions (comment les variables varient ensemble)

Si $\boldsymbol{\Sigma}$ est diagonale, les dimensions sont indépendantes. Sinon, elles sont corrélées, et la "cloche" est inclinée ou étirée dans certaines directions.

### En ML

- **PCA** : trouver les directions de variance maximale, c'est diagonaliser la matrice de covariance
- **VAE** : l'encodeur produit $\boldsymbol{\mu}$ et $\boldsymbol{\Sigma}$ (souvent diagonale) pour chaque point de l'espace latent
- **Gaussian Mixture Models** : chaque composante est une gaussienne multivariée
- **Modèles de diffusion** : le processus de bruitage et débruitage opère dans un espace gaussien multivarié

## Récapitulatif : quelle distribution pour quel usage

| Distribution | Type | Paramètres | Usage en ML |
|---|---|---|---|
| Bernoulli | Discrète (2 valeurs) | $p$ | Classification binaire, dropout |
| Catégorielle | Discrète ($K$ valeurs) | $(p_1, \ldots, p_K)$ | Classification multi-classe, LLM |
| Gaussienne | Continue | $\mu, \sigma^2$ | Initialisation des poids, VAE, diffusion |
| Gaussienne multivariée | Continue ($d$ dim.) | $\boldsymbol{\mu}, \boldsymbol{\Sigma}$ | PCA, espace latent, GMM |

## Ce qu'il faut retenir

- Les modèles de ML produisent et manipulent des distributions de probabilité
- Bernoulli pour le binaire, catégorielle pour le multi-classe, gaussienne pour le continu
- Softmax est le pont entre les logits bruts et une distribution catégorielle valide
- La température contrôle la "certitude" de la distribution softmax
- La gaussienne est omniprésente grâce au théorème central limite et ses propriétés algébriques
- La gaussienne multivariée encode les corrélations entre dimensions via la matrice de covariance
