## Le Chat : l'alternative européenne à ChatGPT

Vous connaissez ChatGPT ? Imaginez la même chose, mais **européenne**, avec un prix **plus attractif**, et vos données stockées en Europe. C'est Le Chat.

Accessible sur chat.mistral.ai, c'est l'interface grand public de Mistral. Vous parlez à une IA sans toucher à une ligne de code. C'est le produit que votre manager, votre comptable ou votre stagiaire peut utiliser sans formation technique.



### Les trois niveaux : du gratuit au corporate

**Free** : gratuit, sans carte bancaire
- Accès à Mistral Small et Medium
- Limites douces sur le nombre de messages par jour
- Recherche web basique
- Historique de conversations sauvegardé
- Applications mobiles iOS et Android

C'est le tier idéal pour les curieux, les étudiants, et tous ceux qui veulent découvrir l'IA conversationnelle sans engagement.

**Pro (14,99 euros/mois)** : le meilleur rapport qualité-prix du marché
- **Mistral Large 3** : le modèle le plus puissant de Mistral
- **Magistral** : le modèle de raisonnement avancé pour les tâches complexes
- Limites de messages élevées
- Canvas (espace de travail collaboratif, voir plus bas)
- Agents personnalisés
- Priorité de service lors des pics de charge

C'est le tier pour les professionnels qui veulent de la puissance IA au quotidien : rédaction, analyse, brainstorming, code.

**Enterprise** : tout ce que Pro offre, plus les fonctionnalités corporate
- Administration des utilisateurs et des équipes
- SSO (Single Sign-On : authentification d'entreprise via SAML/OIDC)
- Politiques de confidentialité et de rétention des données personnalisables
- Support dédié avec SLA garanti
- Conformité RGPD renforcée

Pour les organisations qui déploient l'IA à l'échelle.

<Callout type="info" title="Tarification : Mistral vs OpenAI vs Anthropic">
Le Chat Pro coûte 14,99 euros par mois, contre 20 dollars (environ 18 euros) pour ChatGPT Plus et Claude Pro. Avec la souveraineté européenne des données en bonus, le rapport qualité-prix est difficile à battre.
</Callout>

### Les fonctionnalités qui font la différence

**Recherche web en temps réel** : Le Chat peut chercher sur internet pour vous, en temps réel, et citer ses sources. Indispensable pour les questions d'actualité, les prix, les informations qui évoluent, ou tout sujet postérieur à la date de coupure du modèle.

**Canvas : l'espace de travail collaboratif** : au lieu du va-et-vient classique copier-coller entre le chat et votre éditeur, le Canvas est un **espace de travail intégré** directement dans l'interface. Vous écrivez, Le Chat réécrit ou améliore, vous itérez ensemble sur le même document.

Cas d'usage concrets du Canvas :
- Rédiger un article : Le Chat propose une première version, vous affinez, il reformule
- Générer du code : Le Chat code, vous testez et ajustez directement
- Créer des documents structurés : rapports, présentations, spécifications

<Callout type="tip" title="Canvas vs Chat classique : deux modes complémentaires">
Le chat normal est fait pour les questions-réponses rapides : "Quelle est la capitale du Burkina Faso ?" Le Canvas est fait pour le travail itératif sur un document ou un projet. Les deux modes se complètent selon le besoin.
</Callout>

**Agents personnalisés** : créez votre propre IA sur mesure, configurée pour votre usage spécifique.
- Définissez des instructions système custom (rôle, ton, contraintes)
- Attachez des documents de référence (PDFs, notes, bases de connaissances)
- Configurez les outils que l'agent peut utiliser
- Partagez l'agent avec vos collègues

Exemple concret : créez un agent "assistant juridique français" qui a accès à votre base de contrats types, connaît le droit du travail français, et répond dans un style formel. Vos collègues l'utilisent comme un expert interne.

**Compréhension d'images** : grâce aux capacités multimodales de Pixtral et Mistral Large 3, Le Chat comprend les photos et images.
- Décrire le contenu d'une image
- Extraire du texte (OCR intelligent)
- Analyser un graphique ou un schéma
- Résoudre un problème de maths à partir d'une photo de l'énoncé

### Vie privée : l'avantage européen

C'est l'argument massue de Le Chat face à ChatGPT :

- **Pas d'entraînement sur vos conversations** : par défaut, Mistral n'utilise pas vos échanges pour entraîner ses modèles. Chez OpenAI, il faut explicitement désactiver cette option.
- **Serveurs en Europe** : vos données restent physiquement sur le territoire européen, le RGPD s'applique nativement
- **Suppression à la demande** : vous pouvez supprimer vos données quand vous le souhaitez

Pour les entreprises sensibles à la confidentialité des données, c'est un argument décisif.

<Callout type="warning" title={"Pas de génération d'images nativement"}>
ChatGPT intègre DALL-E pour la génération d'images. Le Chat, à ce jour, n'a pas de générateur d'images intégré. Si la création d'images est un besoin critique pour vous, c'est une limitation à prendre en compte.
</Callout>
