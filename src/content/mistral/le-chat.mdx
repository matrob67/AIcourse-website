## Le Chat : l'alternative européenne à ChatGPT

Vous connaissez ChatGPT ? Imaginez la même chose, mais **européenne**, avec des prix **meilleur marché**, et votre data stockée en Europe. C'est Le Chat.

Accessible via chat.mistral.ai, c'est l'interface grand public de Mistral. Vous parlez à un modèle IA sans toucher à une ligne de code.



### Les trois niveaux

**Free** : gratuit, limites douces
- Accès Mistral Small/Medium
- Limites messages/jour
- Recherche web basique
- Historique gardé
- Smartphones OK

Pour les curieux et les étudiants.

**Pro (14,99€/mois)** : le meilleur rapport qualité-prix
- Mistral Large 3 (le modèle le plus puissant)
- Magistral (raisonnement avancé)
- Limites hautes de messages
- Canvas (voir plus bas)
- Agents perso
- Priorité lors de pic de charge

Pour les pros qui veulent du power au quotidien.

**Enterprise** : tout + facturation centralisée
- Admin des utilisateurs
- SSO (authentification entreprise)
- Politiques données perso
- Support dédié
- SLA garanti
- RGPD renforcé

Pour les organisations.

<Callout type="info" title="Tarifage : Mistral vs OpenAI">
Pro = 14,99€ vs ChatGPT Plus = 20$/mois (~18€). Et vous avez la souveraineté europea en bonus. (Claude Pro = aussi ~20€.)
</Callout>

### Ce que vous pouvez faire

**Recherche web** : Le Chat cherche sur internet en temps réel et cite les sources. Utile pour l'actualité, les prix, les infos fraîches.

**Canvas** : au lieu de copier-coller du texte, le Canvas est un **espace de travail intégré**. Vous écrivez, Le Chat réécrit, vous itérez. Idéal pour :
- Écrire un article (Le Chat propose, vous tweakez)
- Générer du code (Le Chat code, vous testez)
- Créer des documents structurés

C'est du travail collaboratif avec une IA.

<Callout type="tip" title="Canvas vs Chat classique">
Chat normal = questions-réponses rapides
Canvas = travail itératif sur un document. Deux modes complémentaires.
</Callout>

**Agents perso** : créez votre IA sur mesure.
- Instructions système custom
- Attachez des documents (PDFs, notes)
- Configurez les outils utilisables
- Partagez avec collègues

Exemple : créez un agent « assistant juridique français » qui a accès à votre base de contrats.

**Images** : grâce à Pixtral et Mistral Large 3, Le Chat comprend les photos.
- Décrire une image
- Extraire du texte (OCR)
- Analyser un graphique
- Résoudre un problème math dessiné

### Vie privée (l'avantage Europe)

Contrairement à OpenAI :
- **Pas d'entraînement sur vos conversations** (par défaut)
- **Serveurs en Europe** = RGPD appliqué
- Vous supprimez vos données quand vous voulez

C'est un vrai argument pour les entreprises sensibles aux données.

<Callout type="warning" title="Pas de génération d'images nativement">
ChatGPT a DALL-E. Le Chat (à ce jour) n'a pas de générateur d'images. C'est une limite à considérer si les images sont critiques pour vous.
</Callout>
